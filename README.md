# stbimage

Crystal bindings for stb_image

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     stbimage:
       github: crystal-data/stb_image.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "stbimage"
```
